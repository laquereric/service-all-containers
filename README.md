# RayswarmAllContainers

Meta-gem that bundles all Rayswarm container gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'rayswarm-all-containers'
```

## Included Gems

- **magentic-id** — Identity management and WebFinger discovery
- **rayswarm-mermaid** — Mermaid diagram DSL with server-side SVG rendering
- **puml-work** — PlantUML diagram DSL with server-side SVG rendering
- **service-rubygems** — Gem registry engine
- **rayswarm-switch** — Multi-protocol agent communication (MCP, ACP, A2A)

# ServiceAllContainers

Meta-gem that bundles all Service container gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'service-all-containers'
```

## Included Gems

- **magentic-id** — Identity management and WebFinger discovery
- **service-mermaid** — Mermaid diagram DSL with server-side SVG rendering
- **service-puml** — PlantUML diagram DSL with server-side SVG rendering
- **service-rubygems** — Gem registry engine
- **service-switch** — Multi-protocol agent communication (MCP, ACP, A2A)

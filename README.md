# EcosystemsAllContainers

Meta-gem that bundles all ecosystem container gems for convenient inclusion.

## Installation

Add to your Gemfile:

```ruby
gem 'ecosystems-all-containers'
```

## Included Gems

- **service-3d** — 3D data visualization (Three.js + React + Inertia-Rails)
- **service-github** — Git repository hosting and smart HTTP transport
- **service-id** — Identity management and WebFinger discovery
- **service-mermaid** — Mermaid diagram DSL with server-side SVG rendering
- **service-puml** — PlantUML diagram DSL with server-side SVG rendering
- **service-rubygems** — Gem registry engine
- **service-switch** — Multi-protocol agent communication (MCP, ACP, A2A)
- **service-table** — AG Grid table editing engine
